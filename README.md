# fate Controller Remapper & Macro Tool

A personal project for remapping inputs and building custom macros for a game controller (PC / DualSense) using code instead of manufacturer software.

## Overview

This project reads raw controller input and lets you:
- Remap buttons/axes to different outputs
- Create custom macros (multi-button combos, sequences, timed presses)
- Adjust sensitivity, dead zones, and trigger curves
- Save/load different profiles per game
