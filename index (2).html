Read the signal.
Remap the response.

A lightweight PC tool that reads raw input from any controller — Xbox, PlayStation, generic USB/Bluetooth — and turns button presses into keystrokes and macros. No drivers, no console modification, just your OS's own joystick API.

Core

Three moving parts, nothing extra.

01 · remap

Button → key

Bind any controller button straight to a keyboard key. Set it once per game, forget it.

02 · macro

Combo → sequence

Hold two buttons together to fire a scripted sequence of key presses, holds, and timed delays.

03 · profile

Config, not code

Every mapping lives in a JSON profile. Swap games by swapping the file — no code changes.

Profile

A profile is just this.

{
  "controller_index": 0,
  "deadzone": 0.15,
  "button_map": {
    "0": "space",
    "1": "e"
  },
  "macros": {
    "4+5": [["ctrl", "down"], ["c", "tap"], ["ctrl", "up"]]
  }
}
Setup

Running in under a minute.

1
git clone https://github.com/logannanaguemning-code/fate.git && cd fate
2
pip install -r requirements.txt
3
python remapper/main.py --list — confirm your controller shows up
4
python remapper/main.py — starts remapping with the default profile
